# Examples Organization

This directory contains comprehensive NLPL examples organized by topic.

## Directory Structure

### 01_basics/
Fundamental language concepts for beginners
- Basic syntax, variables, functions
- Control flow, loops
- Simple I/O

### 02_oop/
Object-Oriented Programming
- Classes and objects
- Inheritance and interfaces
- Abstract classes
- Design patterns

### 03_type_system/
Type System Features
- Type annotations
- Type inference
- Type guards and aliases
- Variance

### 04_generics/
Generic Programming
- Generic classes and functions
- Type parameters and constraints
- Generic type inference

### 05_functional/
Functional Programming
- Lambda functions
- Higher-order functions
- Functional patterns
- Immutability patterns

### 06_concurrency/
Concurrent and Asynchronous Programming
- Async/await
- Concurrent execution
- Event-driven programming
- Reactive programming

### 07_low_level/
Low-Level System Programming
- Pointers and memory management
- Structs and unions
- FFI (Foreign Function Interface)
- Inline assembly

### 08_networking/
Networking and Web
- Network programming
- Web programming
- Database access

### 09_feature_patterns/
Feature Patterns
- Error handling and logging
- Testing and debugging
- Security and authentication
- Performance optimization
- Dependency injection
- Pattern matching

### 10_stdlib/
Standard Library Examples
- Math, string, collections
- File I/O and system operations
- Complete stdlib demonstrations

## Naming Convention

Files within each directory follow the pattern:
`<number>_<descriptive_name>.nlpl`

Example: `01_hello_world.nlpl`, `02_variables_and_types.nlpl`

## Quick Start Path

New to NLPL? Follow this learning path:
1. `01_basics/01_basic_concepts.nlpl`
2. `02_oop/01_object_oriented.nlpl`
3. `03_type_system/01_type_system_basics.nlpl`
4. `04_generics/01_generic_classes.nlpl`
5. `09_feature_patterns/01_error_handling.nlpl`
