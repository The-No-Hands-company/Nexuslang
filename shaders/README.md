# Shaders Directory

This directory contains shader programs used for testing NLPL's graphics rendering capabilities.

## Contents

- `scene/` - Scene rendering shaders
- `shadow/` - Shadow mapping shaders
- `skybox/` - Skybox rendering shaders

## Purpose

These shaders are used to test:
- NLPL's ability to interface with graphics APIs
- Shader compilation and loading
- Real-time rendering capabilities
- Graphics pipeline integration

## Note

This is part of NLPL's advanced features testing. These shaders demonstrate NLPL's capability to work with low-level graphics systems, though the primary focus of NexusLang is general-purpose programming.
