# Test Textures Directory

This directory contains texture assets used for testing NLPL's graphics and image handling capabilities.

## Contents

- `skybox/` - Skybox texture maps for environment rendering
- `smiley.png` - Test texture for sprite/UI rendering
- `test_pattern.png` - Pattern texture for testing texture mapping

## Purpose

These textures are used to test:

- Image loading and processing
- Texture mapping in graphics applications
- File I/O for binary image formats
- Integration with graphics APIs

## Note

This is part of NLPL's advanced features testing. These assets demonstrate NLPL's capability to handle real-world graphics data, though the primary focus of NexusLang is general-purpose programming with low-level capabilities.
