# NLPL Tutorials

These tutorials teach NLPL programming from first principles through
advanced topics.  Each tutorial is self-contained; you can follow them
in order or jump to the level that suits you.

## Learning Paths

### Beginner Track (~4 hours total)
For programmers new to NLPL, or new to programming altogether.

| # | Tutorial | Time |
|---|---------|------|
| 1 | [Hello World — Your First Program](beginner/01-hello-world.md) | 15 min |
| 2 | [Variables, Functions, and Control Flow](beginner/02-variables-functions-control-flow.md) | 60 min |
| 3 | [Objects and Classes](beginner/03-objects-and-classes.md) | 60 min |
| 4 | [Error Handling](beginner/04-error-handling.md) | 30 min |
| 5 | [Modules and Imports](beginner/05-modules-and-imports.md) | 30 min |

### Intermediate Track (~5 hours total)

| # | Tutorial | Time |

| 2 | [Concurrency Basics](intermediate/02-concurrency-basics.md) | 60 min |
| 3 | [File I/O and Networking](intermediate/03-file-io-networking.md) | 60 min |
